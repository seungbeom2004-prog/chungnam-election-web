# Cute Theme Assets

Place the following files in this directory for the cute layout mode.

## Fonts (`fonts/`)

| File | Description |
|------|-------------|
| `Bingre.woff2` | 빙그레체 Regular (WOFF2 format) |
| `Bingre-Bold.woff2` | 빙그레체 Bold (WOFF2 format, optional) |

## Images (`images/`)

| File | Size | Format | Description |
|------|------|--------|-------------|
| `mascot.png` | ~300x400px | PNG (transparent) | Main mascot character, full body |
| `mascot-small.png` | ~64x64px | PNG (transparent) | Small mascot for buttons/corners |
| `mascot-invite.png` | ~200x200px | PNG (transparent) | Friendly/waving pose for floating invite box |
| `logo-cute.png` | ~64x64px | PNG (transparent) | Party logo in cute style |
| `stars-bg.png` | ~200x200px | PNG (tileable) | Subtle star/sparkle pattern for page background |
| `candidate-frame.png` | ~120x120px | PNG (transparent) | Decorative frame around candidate profile photos on map |
| `speech-bubble.svg` | any | SVG | Reusable speech bubble shape |
| `sparkle.svg` | ~24x24px | SVG | Small sparkle/star decoration |
| `pin-cute.svg` | ~40x40px | SVG | Cute-style map pin base shape |
